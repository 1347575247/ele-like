# 仿饿了么App



# **一、项目介绍**

> 注意：图片、文字等等，这些数据都不是放在数据库里的，都是整理到了本地的一个 json 文件
>
> 中，相当于模拟后台去获取数据。

## **项目技术栈选型**

### **前端技术栈**: vue 3.x、 vite 2.x、 vant UI框架、axios、vue-router 4.x 、vuex 4.x、sass

### **布局：**flexible、flex、rem

### **服务端技术栈：** node+本地 json
